# QTPaint

## About
This is a playground utility to draw things
> [!NOTE]
> This project was made using PyQT6

## How to run or use

> [!IMPORTANT]
> Make sure you have your python and pip installed and added to your PATH

To run this project, clone this repository
```bash
git clone https://github.com/CoreDumpNotCrash/QTPaint.git
```
Go to the root folder
```bash
cd QTPaint
```
Download the dependencies
```bash
pip install -r requirments.txt
```
Run the main file
```bash
python main.py
```
To use this project:
- Hold left button of your mouse to draw
- Hold right button of your mouse to erase
